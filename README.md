kube-top
================

[![CircleCI](https://circleci.com/gh/dpetzold/kube-top.svg?style=svg)](https://circleci.com/gh/dpetzold/kube-top)

Aggregates and provides visual representation of the following kubectl commands:

* `kubectl top pods`
* `kubectl top nodes`
* `kubectl get pods`
* `kubectl get events`
